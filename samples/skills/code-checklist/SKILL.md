# 程式碼檢查表技能 (Code Checklist Skill)

檢查 Python 程式碼時套用此檢查表。

## 程式碼品質檢查表

- [ ] 所有函式都具備類型提示 (type hints)
- [ ] 沒有裸 except 子句 (bare except clauses)
- [ ] 沒有可變預設參數 (mutable default arguments)
- [ ] 檔案 I/O 使用內容管理器 (context managers)
- [ ] 函式少於 50 行
- [ ] 變數與函式名稱遵循 PEP 8 (蛇形命名法 snake_case)

## 輸入驗證檢查表

- [ ] 使用者輸入在處理前經過驗證
- [ ] 已處理邊際情況 (空字串、None、超出範圍的值)
- [ ] 錯誤訊息清晰且有助益

## 測試檢查表

- [ ] 新程式碼有對應的 pytest 測試
- [ ] 已覆蓋邊際情況
- [ ] 測試使用具描述性的名稱

## 輸出格式

將發現結果呈現如下：

```
## 程式碼檢查表：[檔案名稱]

### 程式碼品質
- [通過/失敗] 發現結果的說明

### 輸入驗證
- [通過/失敗] 發現結果的說明

### 測試
- [通過/失敗] 發現結果的說明

### 摘要
有 [X] 個項目在合併前需要注意
```
